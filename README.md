# Amigo Secreto Mobile

### Sobre
Aplicativo desenvolvido em React Native, com o objetiv de possibilitar adicionar uma pessoa a uma lista de participantes de um amigo secreto

<hr />

### Tecnologias
- React Native
- TypeScript
